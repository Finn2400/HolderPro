# HolderPro support

HolderPro is community-supported software provided without warranty.

- Read the [user guide](docs/user-guide.md) and
  [troubleshooting guide](docs/troubleshooting.md).
- Run `holderpro doctor --export holderpro-diagnostics.json` before
  filing an installation or rendering issue.
- Use a [GitHub issue](https://github.com/Finn2400/HolderPro/issues/new/choose)
  for reproducible defects and focused feature requests.
- Use [GitHub Discussions](https://github.com/Finn2400/HolderPro/discussions)
  for workflow questions and print-design advice when Discussions is enabled.
- Report vulnerabilities privately using [SECURITY.md](SECURITY.md).

Diagnostic bundles omit model geometry and redact user paths by default. Check
the archive yourself before uploading it. Never upload a model you do not have
permission to share.
